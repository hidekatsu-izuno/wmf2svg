# wmf2png Paint White Background

## Purpose
Make `src/test/bin/wmf2png.ps1` render PNG output closer to Microsoft Paint when converting WMF files whose untouched canvas area appears white in Paint-generated PNGs.

## Context
- The current script has two rendering paths:
  - GDI+ `Image.FromFile` / `Graphics.DrawImage` / `Bitmap.Save`.
  - Non-placeable WMF `SetMetaFileBitsEx` / `PlayMetaFile` / DIB section / GDI+ PNG encoder.
- The user observed that Paint-generated `2doorvan.png` previews transparent regions as white, while PNGs generated by `wmf2png.ps1` preview those areas as black.
- The user also observed that Paint emits transparent black for WMFs that have a placeable header.
- The script should therefore preserve transparent black for placeable WMFs and use transparent white only for non-placeable WMFs where Paint does so.
